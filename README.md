# SaitMinisterstvoPoDelamMolodeji

🌟 Молодежная платформа волонтерства

Веб-приложение для организации волонтерской деятельности, управления мероприятиями и учета часов волонтеров.

📋 О проекте

Платформа предоставляет удобный интерфейс для:
— **Волонтеров**: регистрация на мероприятия, просмотр своего профиля, отслеживание накопленных часов
— **Администраторов**: создание и управление мероприятиями, учет часов волонтеров, генерация отчетов, управление бюджетом

Проект разработан на Flask с использованием SQLite в качестве базы данных.

🚀 Основной функционал

Для волонтеров
— Регистрация и авторизация по номеру телефона
— Восстановление пароля через кодовое слово
— Просмотр предстоящих мероприятий
— Регистрация на мероприятия (как участник или волонтер)
— Личный кабинет с историей участия
— Загрузка аватара и заполнение профиля
— Просмотр накопленных волонтерских часов

Для администраторов
— Панель управления с общей статистикой
— CRUD операции с мероприятиями
— Загрузка фото и материалов для мероприятий
— Управление регистрациями участников и волонтеров
— Начисление часов волонтерам (с возможностью штрафов)
— Завершение мероприятий с автоматическим начислением часов
— Генерация отчетов в формате Word:
  — Полный отчет по мероприятию
  — Список участников
  — Список волонтеров
  — Общий список всех волонтеров
— Учет бюджета мероприятия (чек-лист закупок)

 🛠 Технологии

— Backend: Python 3, Flask
— Database: SQLite, SQLAlchemy ORM
— Authentication: Flask-Login
— File handling: Werkzeug
— Document generation: python-docx
— Frontend: HTML, CSS, JavaScript (Bootstrap)

 🗄 Модели базы данных

— User - пользователи (волонтеры и администраторы)
— Event - мероприятия
— EventRegistration - регистрации на мероприятия
— VolunteerHours - учет волонтерских часов
— Notification - уведомления
— BudgetItem - элементы бюджета
— File - файлы

🔧 Установка и запуск

1. Клонировать репозиторий
```bash
git clone https://github.com/yourusername/youth-volunteer-platform.git
cd youth-volunteer-platform
```

2. Создать виртуальное окружение
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```

3. Установить зависимости
```bash
pip install flask flask-sqlalchemy flask-login werkzeug python-docx
```

4. Запустить приложение
```bash
python app.py
```

При первом запуске автоматически:
— Создадутся все необходимые таблицы в БД
— Создастся администратор: `+7(928)031-93-29` / пароль: `7777`
— Создастся тестовый волонтер: `+7(900)111-22-33` / пароль: `12345`

Приложение будет доступно по адресу: `http://localhost:5000`

 👥 Тестовые учетные записи

| Роль | Телефон | Пароль | Кодовое слово |
|------|---------|--------|---------------|
| Администратор | +7(928)031-93-29 | 7777 | admin |
| Волонтер | +7(900)111-22-33 | 12345 | volonter |

 📄 Форматы отчетов

— `report_{название_мероприятия}_{дата}.docx` - полный отчет по мероприятию
— `participants_{название}_{дата}.docx` - список участников
— `volunteers_{название}_{дата}.docx` - список волонтеров мероприятия
— `all_volunteers_{дата}.docx` - общий список всех волонтеров

 🔐 Особенности безопасности

— Хеширование паролей (Werkzeug)
— Защита маршрутов через `@login_required`
— Разграничение прав доступа (admin/volunteer)
— Защита от CSRF (через Flask-WTF в формах)
— Безопасное имя файлов через `secure_filename`
